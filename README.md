# SA Church Finder

Airbnb-style church discovery app for San Antonio.

## Tech Stack

- **Frontend:** React 18 + TypeScript + Vite + Tailwind CSS + shadcn/ui
- **Backend:** Node.js + Express + Prisma + PostgreSQL with PostGIS
- **Maps:** Mapbox GL JS
- **Auth:** Passport.js (local + Google OAuth)
